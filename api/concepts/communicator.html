communicator CCL_API split_communicator (const communicator &comm, int color, int key)

Creates a new sub-communicator from an existing communicator.

Creates one or more new communicators from an existing communicator. All ranks providing the same color will be placed in the same sub-communicator. Ranks within each sub-communicator are ordered according to the key.

Parameters:

  • comm – existing communicator from which the new subcommunicator(s) will be derived

  • color – determines which sub-communicator the calling rank is assigned to. Ranks with the same color will be on the same sub-communicator. If the color is different, the ranks will be on different sub-communicators.

  • key – defines the rank ordering within the new sub-communicator. Ranks in the new communicator are sorted by ascending key.

Returns:

the newly created sub-communicator
